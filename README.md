# Python Discord-bot
Python discord bot for Music/sound effects in vc

1.Put Discord bot token and spotify token in a .env like this (do NOT share any tokens!)
TOKEN=
SPOTIFY_CLIENT_ID=
SPOTIFY_CLIENT_SECRET=

2.Put all necessary bot perms and invite the bot to serv

3.Customise by replacing the name "Discordbot" in discordbot.py

4.Install dependencies

"python -m venv venv"
"venv\Scripts\activate"
"pip install -U pip"
"pip install -U discord.py PyNaCl python-dotenv requests yt-dlp Pillow deezer-python davey"
